🚀 Python Project | Advanced E-commerce Shopping Cart System

Developed a comprehensive E-commerce Shopping Cart System in Python featuring product management, inventory tracking, secure payment processing, discount handling, loyalty programs, and smart product recommendations. The project simulates a real-world online shopping platform with advanced cart operations and order management functionalities.

💡 Key Features:
• Product catalog with inventory management
• Shopping cart with dynamic pricing and discounts
• Secure payment processing (Card & PayPal simulation)
• Customer loyalty and tier-based pricing system
• Product reviews and rating system
• Order history and checkout management
• Product search and category filtering
• Data persistence using JSON storage

This project helped me strengthen my understanding of Python, Object-Oriented Programming (OOP), file handling, payment workflow simulation, and enterprise-level application design.
